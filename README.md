## Escenario Elegido

**Escenario B - Análisis de Ventas de una Pequeña Empresa**

Análisis de dataset de ventas para generar indicadores clave de desempeño comercial.

## Estructura del Repositorio

- /datos: Dataset de ventas en formato CSV
- /scripts: Script Python de análisis
- /resultados: Gráficos y resultados generados

## Dataset Utilizado

Dataset de ventas simuladas con registros diarios de fecha y monto de venta.
Fuente: datos abiertos de acceso público.

## Instrucciones para Ejecutar

1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar scripts/analisis_ventas.py
4. Los resultados se guardan en /resultados
